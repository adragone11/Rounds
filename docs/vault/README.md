# Pip Knowledge Vault

Open this folder as a vault in Obsidian: **File > Open Vault > Open folder as vault** → select `docs/vault`

## Structure

- **daily/** — Session logs (what was built, decisions made)
- **research/** — Competitor analysis, feature research, user feedback
- **architecture/** — System design, data flow, tech decisions
- **issues/** — Bugs found, root causes, fixes applied
- **insights/** — Things that worked, didn't work, lessons learned
